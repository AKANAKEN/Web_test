Panduan Mengubah Tampilan Live Chat pada OBS Menggunakan CSS
               
    
                    
1. Menambahkan Live Chat ke OBS

                    
Sebelum mengubah tampilan dengan CSS, pastikan Anda sudah menambahkan live chat ke OBS dengan cara berikut:

                    
    
                        
  1. Buka OBS Studio.
    2. 
                        
  2. Klik tombol + di Sources dan pilih Browser.
    3. 
                        
  3. Masukkan URL live chat dari platform yang digunakan (YouTube, Twitch, dll.).
    4. 
                        
  4. Atur ukuran sesuai kebutuhan, lalu klik OK.
    5. 
                    

                    
                    
2. Menggunakan CSS untuk Mengubah Tampilan Live Chat

                    
Setelah menambahkan live chat, Anda bisa mengubah tampilannya menggunakan CSS.

                    
                    
a. Membuka Pengaturan CSS di OBS

                    
    
                        
  1. Klik kanan pada sumber Browser yang berisi live chat.
    2. 
                        
  2. Pilih Properties.
    3. 
                        
  3. Di bagian bawah, temukan kotak teks untuk Custom CSS.
    4. 
                    

                    
                    
b. Menyesuaikan Tampilan Live Chat dengan CSS

                    
Tempelkan kode CSS yang didapatkan sebelumnya pada Web SkewFold ke dalam kotak Custom CSS:

                    
Berikut Contoh CSS nya

                    
                    
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@100;200;300;400;500;600;700;800;900&display=swap');
                        @import url('https://fonts.googleapis.com/css2?family=DM+Serif+Display&family=M+PLUS+Rounded+1c:wght@100;300;400;500;700;800;900&display=swap');
                        
                        :root {
                          /* chat */
                          /* owner/streamer */
                          --font-color-author-name-owner: #4a4355;
                          --background-author-name-owner: #ffffff;
                          --font-color-chat-owner: #ffffff;
                          --background-chat-owner: #F9A825;
                        
                        
                          /* moderator */
                          --font-color-author-name-moderator: #4a4355;
                          --background-author-name-moderator: #ffffff;
                          --font-color-chat-moderator: #ffffff;
                          --background-chat-moderator: #1D77CA;
                        
                          /* member */
                          --font-color-author-name-member: #ffffff;
                          --background-author-name-member: #00BF71;
                          --font-color-chat-member: #4a4355;
                          --background-chat-member: #ffffff;
                        
                          /* viewers */
                          --font-color-author-name-viewers: #ffffff;
                          --background-author-name-viewers: #1D77CA;
                          --font-color-chat-viewers: #4a4355;
                          --background-chat-viewers: #ffffff;
                        
                          /* membership announce */
                          --font-color-membership: #00BF71;
                          --font-color-sub-text-membership: #00BF71;
                          --font-color-other-text-membership: #4a4355;
                          --font-color-chat-membership: #00BF71;
                          --background-membership: #ffffff;
                        }
                        

                    
                    
c. Menerapkan dan Menyesuaikan

                    
    
                        
  1. Klik OK setelah menempelkan kode CSS.
    2. 
                        
  2. Cek tampilan di OBS.
    3. 
                        
  3. Sesuaikan CSS sesuai dengan preferensi.
    4. 
                    

                    
                    
3. Tips Tambahan

                    
    
                        
  • Jika live chat memiliki elemen berbeda (misalnya YouTube vs Twitch), gunakan Inspect Element di browser untuk menemukan class yang ingin diubah.
    • 
                        
  • Gunakan opacity: 0; jika ingin menyembunyikan elemen tanpa menghapusnya.
    • 
                        
  • Coba berbagai kombinasi warna dan font untuk tampilan yang lebih menarik.
    • 
                    

                    
                    
Dengan langkah-langkah ini, Anda bisa menyesuaikan tampilan live chat agar lebih sesuai dengan tema siaran Anda di OBS. Selamat mencoba!
